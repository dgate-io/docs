`dgate-cli` is a command line interface for DGate (API Gateway) Admin API.

## Usage

```
dgate-cli [global options] command [command options] 
```

## Version

```
dev
```

## Commands

- `namespace, ns`: Namespace commands
- `service, svc`: Service commands
- `module, mod`: Module commands
- `route, rt`: Route commands
- `domain, dom`: Domain commands
- `collection, col`: Collection commands
- `document, doc`: Document commands
- `help, h`: Shows a list of commands or help for one command

## Global Options

- `--admin value`: The URL for the file client (default: "http://localhost:9080") [$DGATE_ADMIN_API]
- `--auth value, -a value`: The authorization credentials or token [$DGATE_ADMIN_AUTH]
- `--help, -h`: Show help
- `--version, -v`: Print the version

## Namespace Commands

### dgate-cli namespace

```
dgate-cli namespace command [command options] [arguments...]
```

#### Commands

- `create, mk`: Create a namespace
- `delete, rm`: Delete a namespace
- `list, ls`: List namespaces
- `get`: Get a namespace
- `help, h`: Shows a list of commands or help for one command

#### Options

- `--help, -h`: Show help
